Scenarios

Selected scenarios of physiology and pathophysiology powered by Physiolibrary and Bodylight technology in single or multipage web-simulators. See at Bodylight-Scenarios.

VirtualBody

WebGL powered visualisation of human body with simplified anatomy view and selected interactive web simulators. Can be viewed on all modern browser (chrome 88.x, firefox 86.x, ms edge 88.x, safari 14.x) on desktop, table or mobile device. Additionally, Text2Speech api is demonstrated for reporting the status of virtual patient. See demo version at Bodylight-VirtualBody.

Editor

A web simulator using Bodylight.js2.0 technology can be written in any HTML or Markdown editor. However, to facilitate Special Bodylight Editor integrates generic ACE editor with markdown-it renderer to a static web application allowing to edit and live-preview web simulators. Use it at Bodylight-Editor.

Tutorial

Tutorial to create simple and advanced web simulator from Modelica model source code to functional interactive web simulator using Bodylight toolchain (Bodylight-.js FMU Compiler, Editor, Bodylight.js-Components). Following the tutorial, you will get set of files in HTML,JS and MARKDOWN format ready to be deployed as a static web page.

1.0 Composer

A web simulator using Bodylight.js1.0 technology can be created only with Composer, it allows you to visually create an HTML simulator. We are using the wonderful project GrapesJS to provide the layouting engine, Adobe® Animate to create interactive animations and Modelica to make it all run on models your write yourself..
  • Use it at Bodylight-Composer.
  • Tutorial for Composer at docs/introduction.

1.0 Physiome Apps

Set of web simulator developed in Bodylight.js1.0 technology include simple non-pulsatile circulation simulator, Nephron kidney simulator and simulator of Iron Metabolism.
